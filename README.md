# Java-Bank-Management-System
Java Swing-based Bank Management System with authentication, core banking operations, email notifications and file persistence

Java Bank Management System

A Java-based desktop banking application developed as an OOP Final Project
simulating real-world banking operations through a structured GUI.

## Features
- Separate Client and Employee modules with secure authentication
- Account creation with strong validation and uniqueness checks
- Deposit, Withdrawal and Fund Transfer
- Transaction logging and Statement of Account (SOA) generation
- Automated Email notifications via JavaMail API
- Bank asset monitoring and account management
- File-based data persistence
- Clean GUI with smooth navigation and animations

## Tech Stack
Java | Java Swing | OOP | File Handling | JavaMail API
